<h3>How to run 🚀</h3>
<p>
    To run the bot use the next commands in terminal
</p>
<h4>On Windows</h4>
<pre>
<code>virtualenv venv
venv\Scripts\activate
python src\bot.py</code>
</pre>
<h4>On Linux/Mac</h4>
<pre>
<code>virtualenv venv
. venv/Scripts/activate
python src/bot.py</code>
</pre>
